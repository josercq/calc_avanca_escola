# Calculadora

Calculadora em HTML para uso institucional.

**Descrição:**
- Projeto simples que fornece uma interface de calculadora baseada em `index.html`.
- Destina-se a demonstração e uso local em navegadores modernos.

**Funcionalidades (típicas):**
- Operações básicas: adição, subtração, multiplicação e divisão.
- Interface minimalista pronta para uso imediato.

**Como executar**
- Método rápido: abra o arquivo `index.html` diretamente no navegador (duplo-clique ou arraste para o navegador).

- Servir localmente (opcional, caso prefira um servidor local). Exemplo com Python:

```
# Em PowerShell (no diretório do projeto):
python -m http.server 8000
# Depois abra http://localhost:8000 no navegador
```

**Requisitos**
- Navegador moderno (Chrome, Edge, Firefox, Safari).
- Não há dependências externas.

**Estrutura do projeto**
- `index.html` : interface e lógica da calculadora.
- `README.md`  : este arquivo.
